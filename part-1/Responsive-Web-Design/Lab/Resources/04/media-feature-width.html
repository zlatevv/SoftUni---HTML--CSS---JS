Media Feature: width, min-width, max-width

See also: https://developer.mozilla.org/en-US/docs/Web/CSS/Media_Queries/Using_media_queries

width

Targets viewports that are exactly the specified width.

@media (width: 800px) {
    body::before {
        display: block;
        content: 'The viewport width is exactly 800px';
    }
}

min-width

Targets viewports with a width equal to or greater than the specified value.

@media (min-width: 800px) {
    body::before {
        display: block;
        content: 'The viewport width is larger than 800px';
    }
}

max-width

Targets viewports with a width equal to or less than the specified value.

@media (max-width: 800px) {
    body::before {
        display: block;
        content: 'The viewport width is smaller than 800px';
    }
}
